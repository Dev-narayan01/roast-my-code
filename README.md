# 🔥 Roast My Code

> Paste your code. Get destroyed by AI. No mercy.

**Live Demo → [roast-my-code-lemon.vercel.app](https://roast-my-code-lemon.vercel.app)**

---

## What is this?

Roast My Code is an AI-powered code reviewer that brutally critiques your code with dark humor and savage honesty. Submit any code snippet, choose your pain level, and watch an LLM tear it apart — technically and mercilessly.

## Features

- **Three roast levels** — Mild, Medium, and Brutal
- **AI-powered analysis** — uses Groq's LLaMA 3.1 for fast, sharp roasts
- **Code score** — get rated 1–10 (most code deserves below 5)
- **Backhanded compliment** — because every roast needs one redeeming moment
- **Supports any language** — Python, Java, C, JavaScript, whatever you wrote at 3am

## Tech Stack

- **Frontend** — React + TypeScript + Vite
- **Styling** — Tailwind CSS
- **AI** — Groq API (LLaMA 3.1 8B Instant)
- **Deployment** — Vercel

## Running Locally

```bash
git clone https://github.com/Dev-narayan01/roast-my-code.git
cd roast-my-code
npm install
```

Create a `.env` file:

```
VITE_GROQ_API_KEY=your_groq_api_key
```

Get a free Groq API key at [console.groq.com](https://console.groq.com).

```bash
npm run dev
```
